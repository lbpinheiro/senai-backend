Prova Back End Sênior

Exercício para o Processo Seletivo

Endpoint Método Descrição JSON esperado (entrada)
/api/0.1/partido POST Cadastra um novo partido

nome: String

numero: Number
/api/0.1/politico POST Cadastra um novo político

nome: String

cpf: Number

rg: Number

dataNascimento: Date

idPartido: Number
/api/0.1/eleicao POST Cadastra uma nova eleição

ano: Number

pais: String

cargo: String [Prefeito, Governador, Senador, Presidente]
/api/0.1/candidato POST Faz a associação Político-Eleição

idPolitico: Number

idEleicao: Number

Sobre a API

  • Banco de dados

    Mongodb, o arquivo de configuração .env como padrão está apontando para uma base local

  • Server

    node.js, para rodar o site basta o comando 'npm start' no mesmo diretório do package.json

JSON de retorno

  • Execução bem sucedida: { "msg": [String] }
  • Execução com erro: { "msg": [String], "errorCode": [String] }
